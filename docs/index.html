Quick Start

Installation

Get started instantly with npx (recommended):

npx insite-mcp

Or install globally:

npm install -g insite-mcp

MCP Configuration

Add to your MCP client configuration (npx - recommended):

{
  "mcpServers": {
    "insite": {
      "command": "npx",
      "args": ["insite-mcp"]
    }
  }
}
